DESAFIO — Gerenciador de Biblioteca (Java)

Crie um programa em Java executado no terminal que permita:

- Cadastrar livros (título e autor)
- Listar livros cadastrados
- Buscar livros pelo título
- Menu interativo

Entrada e saída via terminal.
